Class OngoingKaijuJobsInfo

  • All Implemented Interfaces:
    
public class OngoingKaijuJobsInfo

    It used to keep in memory the pending jobs in Kaiju side. This object is saved in the Shared preferences in case of the app has been destroyed when the whole of jobs hasnt been treated before the destruction of the app. So when the app is restarted, the OngoingKaijuJobsInfo is read by KaijuJobsWatchdog.
